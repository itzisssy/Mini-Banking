# 🏦 Python Mini-Banking System (Basic + Advanced)

Dieses Projekt enthält zwei Versionen eines einfachen Bankprogramms in Python:

- 🔹 Datei 1: Basic Version (einfach)
- 🔹 Datei 2: Advanced Version (mit Speicherung & Menü-System)

---

# 📌 BASIC VERSION

Ein sehr einfacher Bankautomat ohne Speicherung.

## 💡 Funktionen

- Einzahlen
- Abheben
- Kontostand anzeigen

---

## ▶️ Menü

a) Einzahlen  
b) Auszahlen  
c) Kontostand abfrage  

---

## 💡 Beispiel

Wählen Sie eine von den drei Optionen: a  
Wie viel wollen sie einzahlen?  
Summe: 50  
Auf dein Konto ist jetzt 50.0 Euro  

---

## ⚠️ Hinweise

- Kontostand wird nicht gespeichert
- Nach Programmende geht alles verloren
- Sehr einfache Version

---

# 📌 ADVANCED VERSION

Ein verbessertes Banking-System mit:

- Speicherfunktion
- Endlosschleife
- Fehlerprüfung
- Datei-Speicherung

---

## 🧠 Features

- Kontostand wird in `kontostand.txt` gespeichert
- Automatisches Laden beim Start
- Schutz vor falschen Eingaben
- Abheben nur möglich, wenn genug Geld vorhanden ist
- Programm kann beendet werden

---

## ▶️ Menü

a) Einzahlen  
b) Auszahlen  
c) Kontostand abfragen  
d) Beenden  

---

## 💡 Beispiel

--- Mini-Banking ---  
a) Einzahlen  
b) Auszahlen  
c) Kontostand abfragen  
d) Beenden  

Wählen Sie eine Option: a  
Wie viel wollen Sie einzahlen? 100  
Auf dein Konto ist jetzt 100.0 Euro  

---

## 💾 Speicherung

Beim Beenden wird der Kontostand automatisch gespeichert:

kontostand.txt

Beim nächsten Start wird der letzte Kontostand geladen.

---

## ⚠️ Fehlerbehandlung

- Ungültige Eingaben werden abgefangen
- Abheben nur möglich bei genug Guthaben
- Datei wird sicher gespeichert
